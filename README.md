# OpenBlock External Resources - Vietnamese Patch

Repo này là bản chỉnh sửa (patch) external resources của OpenBlock
nhằm Việt hoá một phần giao diện và tối ưu tài nguyên.

## Phạm vi
- Chỉnh sửa nội dung hiển thị (text / ngôn ngữ)
- Điều chỉnh một số block cho phù hợp nhu cầu sử dụng
- Không cung cấp source code build OpenBlock
- Không hỗ trợ build lại hoặc đóng gói cho Linux

## Lưu ý
Project này không phải repo phát triển chính thức,
chỉ cung cấp bản đã chỉnh sửa để sử dụng nhanh.
